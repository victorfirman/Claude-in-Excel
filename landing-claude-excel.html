Prerequisite

Dua hal yang perlu disiapkan agar bisa menggunakan Claude langsung di dalam Microsoft Excel.

Paket ClaudeBerlangganan berbayar

  • Minimal Claude Pro (US$20/bulan)
  • Tersedia juga di Max, Team & Enterprise
  • Paket Free tidak didukung

Microsoft ExcelLewat Microsoft 365

  • Excel Web, Windows (M365), Mac & iPad
  • Akun Microsoft 365 kerja / sekolah · file .xlsx / .xlsm
  • Tidak didukung: Excel 2016/2019, Android, akun personal @outlook
Cara Aktivasi 1Install add-in dari Microsoft Marketplace → 2Sign in akun Claude → 3Buka file Excel
