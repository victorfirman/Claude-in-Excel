## ✅ Prasyarat memakai Claude in Excel

- **Paket Claude:** minimal **Pro** (juga Max, Team, Enterprise) — Free tidak didukung.
- **Excel via Microsoft 365:** Web, Windows (M365), Mac, iPad + akun kerja/sekolah;
  file `.xlsx` / `.xlsm`. Tidak didukung: Excel 2016/2019, Android, akun personal @outlook.
- Install add-in dari Microsoft Marketplace → sign in akun Claude → buka file Excel.

> Sumber: [Use Claude for Excel — Claude Help Center](https://support.claude.com/en/articles/12650343-use-claude-for-excel)
